Убедитесь, что на устройстве установлен python 3.9+ , к интерпритатору подключены библиотеки "pygame" и "easygui"
Для использования программы перейдите в деррикторию "продукт" и запустите файл "main.py"

Инструкция:
Для приближения/отдаления игрового поля крутите колесико мыши.
Для увеличения/уменьшения коэф. времени крутите колесико мыши с зажатым CTRL.
для добавления нового объекта нажмита ПКМ.
Для перемещения по игровому полю зажмите ЛКМ и "оттягивайте" экран в нужную сторону.
Для постановки программы на паузу нажмите SPACE ли зайдите в настройки.
Чтобы зайти в настройки / выйти из них нажмите на шестеренку в правом верхнем углу.
В меню настроек вы можете увеличивать/уменьшать радиус и массу тела, которое вы создаете на игровом поле,
кроме того, вы можете выбрать другой цвет для этого тела, изменить ограничение по FPS.
В меню настроек можно выбрать новый или сохранить текущий шаблон, который хранит в себе информацию об объектах на поле.
